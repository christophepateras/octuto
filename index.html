git

dans le répertoire local: git init

git add *

git commit - m "message"

prendre l'adresse https sur le site de github dans le projet ("onglet code et bouton code")

git remote add origin [adresse https]

(pour connaitre celle déjà active : git remote -v)

pour enlever adresse : git remote rm [origin]

pour forcer : git push -f origin main

git branch -M main

git push -u origin main
